Replaced

The standalone Airport Rag map has been retired.

The interactive 3D airport map is now part of the Powermind 2 companion app, served from ../powermind2, and the standalone React demo at ../src.

All airport data (gates, shops, restrooms, security, roadways) now lives in powermind2/src/map/data/airportData.js and is the single source of truth for the application.

Run npm install && npm run dev inside powermind2 or src to launch the new map.
